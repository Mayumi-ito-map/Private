正規表現のデフォルト動作（貪欲マッチ）は、条件に合う限りできるだけ長く文字列を取り込もうとします。
これに対し、非貪欲マッチは**最初に見つけた終了条件でマッチを停止**します。 
記述方法
量指定子（繰り返しを表す記号）の**直後に ?**を付けます。 

種類 	貪欲（デフォルト）	非貪欲（最短一致）
0回以上	*	*?
1回以上	+	+?
0回か1回	?	??
指定回数以上	{n,}	{n,}?

#カンマ並び替え
def reorder_by_comma(text: str):
    if "," not in text:
        return [text]      # ← 必ず list

    parts = [p.strip() for p in text.split(",")]
    if len(parts) != 2:
        return [text]    # ← 必ず list

    return [f"{parts[1]} {parts[0]}"]

分割した結果、要素が3つ以上（例: "姓, 名, 称号") ある場合や、1つしかない場合は、処理の対象外として元のテキストを返します。    
2番目の要素（名）を前に、1番目の要素（姓）を後ろに配置し、間にスペースを入れて1つの文字列にします。
